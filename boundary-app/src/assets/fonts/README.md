# Font Files Directory

This directory should contain the Gilroy font files for the application.

## Required Files:
- Gilroy-Regular.ttf
- Gilroy-Medium.ttf  
- Gilroy-SemiBold.ttf
- Gilroy-Bold.ttf

## Instructions:
1. Download the Gilroy font Circle
2. Copy the required .ttf files to this directory
3. Follow the setup guide in `mobile/GILROY_FONT_SETUP.md`

## Note:
The application is already configured to use these fonts. Once the font files are added, restart the development server to see the changes.
